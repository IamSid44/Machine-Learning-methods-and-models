# Classification

This folder contains supervised learning notebooks focused on image and text classification.

## Notebooks

- [01_multitask_cnn_fmnist.ipynb](01_multitask_cnn_fmnist.ipynb)
  - Multi-task CNN that jointly solves classification and regression on Fashion-MNIST.
- [02_image_colorization_cifar10.ipynb](02_image_colorization_cifar10.ipynb)
  - Encoder-decoder style workflow for image colorization on CIFAR-10.
- [03_decision_trees_text.ipynb](03_decision_trees_text.ipynb)
  - Decision tree pipeline for sentiment prediction from text-derived features.

## Notes

- These notebooks may involve heavier training and sweeps.
- Saved outputs are retained to keep the repository showcase-friendly.
