# Clustering

This folder contains unsupervised learning notebooks, from hard clustering to probabilistic clustering.

## Notebooks

- [01_kmeans_clustering.ipynb](01_kmeans_clustering.ipynb)
  - K-Means from scratch with elbow and silhouette-based model selection.
- [02_gaussian_mixture_models.ipynb](02_gaussian_mixture_models.ipynb)
  - GMM from scratch using EM, with BIC and silhouette comparisons.
- [03_image_segmentation_gmm.ipynb](03_image_segmentation_gmm.ipynb)
  - Color-based image segmentation using GMM, plus convergence visualization.

## Notes

- These notebooks are generally lighter than CNN-based tasks.
- Results are retained with plots and summary cells for quick review.
